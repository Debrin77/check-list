⚠️ Castelló de la Plana: El agua del grifo tiene EC ~850 µS/cm (agua muy dura). Solo quedarían ~550 µS/cm de margen para nutrientes, insuficiente para lechugas. Se recomienda encarecidamente usar agua destilada u ósmosis.

📖 Qué buscar en el SINAC
1. Conductividad — el más importante. < 400 µS/cm = buena, < 600 = aceptable, > 700 = problemática para hidropónica
2. Bicarbonatos (HCO₃) — > 200 mg/L sube mucho el pH, necesitarás más pH-
3. Sodio (Na) — > 50 mg/L dificulta absorción de calcio y potasio
4. Cloro residual — > 0.5 mg/L puede afectar a las raíces. Dejar reposar 24h o usar vitamina C
Cómo llegar: sinac.sanidad.gob.es → Acceso ciudadano → Información actual → Elige provincia y municipio → Busca "Conductividad" en la tabla de parámetros
